# 🎤 Speech-to-Text AI Web Application

An AI-powered **Speech-to-Text Web Application** that converts spoken audio into text using the OpenAI Whisper model.

The application allows users to **upload audio files or record audio using a microphone** and converts speech into text using advanced Natural Language Processing (NLP).

---

## 🚀 Features

- 🎤 Live microphone recording
- 📁 Upload audio files
- 🧠 AI-powered speech recognition using Whisper
- ⚡ Fast transcription
- 🎧 Audio preview before transcription
- 🌐 Modern responsive web interface
- 🔄 Flask backend integration
